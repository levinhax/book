Git Commit 代码提交规范

规范名 | 描述
---|---
init | init
version | 发布版本
feat | 新增feature
fix | 修复bug
merge | 合并分支
docs | 仅仅修改了文档，README，CHANGELOG，CONTRIBUTE等
style | 仅仅修改了空格，格式缩进，逗号，样式等，不改变代码逻辑
test | 测试用例，包括单元测试、集成测试等
chore | 改变构建流程、或者增加依赖库、工具等
perf | 优化相关，比如提升性能/体验
refactor | 代码重构，没有增加新功能或者修复bug
revert | 回滚到上一个版本
